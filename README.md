# My Dotfiles

## Installation

### Install

1. Install XCode
2. Install [iTerm 2](https://www.iterm2.com/)
3. Install [Oh My Zsh](https://github.com/robbyrussell/oh-my-zsh)

### Bootstrap

```bash
./bootstrap.sh
```
